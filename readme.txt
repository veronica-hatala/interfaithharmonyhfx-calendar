=== Interfaithharmonyhfx Calendar ===
Contributors: Veronica Hatala

A calendar plugin for Interfaith Harmony Halifax.

== Description ==

Create a date sorted calendar for IHH.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/interfaithharmonyhfx-calendar` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
